# Videos Folder

## Purpose
Store promotional videos, project demos, or background videos here.

## Supported Formats
- MP4 (recommended)
- WebM
- OGG

## Best Practices
1. Keep file sizes under 10MB for web performance
2. Use compressed/optimized videos
3. Consider using YouTube/Vimeo embeds for longer videos
4. Use descriptive filenames: `project-demo.mp4`, `intro-video.mp4`

## How to Use
Reference videos in your components:
```jsx
<video src="/assets/videos/your-video.mp4" />
```
