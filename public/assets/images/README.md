# Images Folder

## Required Images

### Profile Photo
- **Filename**: `profile.jpg` or `profile.png`
- **Recommended Size**: 500x500px (square aspect ratio)
- **Usage**: About Me section
- **Tips**: Use a professional headshot with good lighting

### Project Screenshots (Optional)
Place project images here with descriptive names:
- `project-chatbot.png`
- `project-security-tool.png`
- etc.

### Other Images
Add any other images for the portfolio here.

## How to Add Your Photo

1. Take or select a professional photo
2. Crop it to a square (1:1 aspect ratio)
3. Resize to 500x500px
4. Save as `profile.jpg` or `profile.png`
5. Place it in this folder: `public/assets/images/`

The About section will automatically display your photo.
